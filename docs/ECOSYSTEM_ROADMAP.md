# Privacy-First Ecosystem Roadmap

**Mission:** Build tools for seniors that respect privacy while delivering
functionality families need.

## Phase 1: Foundation (2026)

### Q3 2026: Shopping List + Meal Planner + Pantry Tracker

- [ ] App launch
- [ ] Server-side encryption at rest
- [ ] Row-level security (users see only their data)
- [ ] Family sharing (explicit permission model)
- [ ] Biometric unlock (Face ID / Touch ID)
- [ ] Data export (JSON)
- [ ] Annual privacy audit

## Phase 2: Family Features (2026-2027)

### Q4 2026: Medication Tracker

- [ ] Same privacy model as Phase 1
- [ ] Caregiver can see medication reminders (no other data)
- [ ] No automatic sync to third-party health platforms
- [ ] HIPAA-compliant (if handling health data)

### Q1 2027: Health Metrics Dashboard

- [ ] Vitals tracking (blood pressure, weight, glucose)
- [ ] No sharing to insurance companies or advertisers
- [ ] Optional: Share with user's doctor (with explicit consent)

### Q2 2027: Appointment & Caregiver Calendar

- [ ] Shared family calendar (births, appointments, tasks)
- [ ] Notification preferences (which family members see what)
- [ ] No sync to Google Calendar / Outlook without explicit user consent

## Phase 3: Privacy Control Center (2027)

### Q2-Q3 2027: Account Dashboard

- [ ] One place to see all apps + your data access
- [ ] Revoke app permissions instantly
- [ ] Download all data across all apps (bulk export)
- [ ] Optional: Self-hosting option (run your own server)

## Phase 4: Ecosystem Expansion (2027+)

### Optional Future Apps

- **Financial Overview:** Simplified bill tracking, net worth (no integration
  with banks without consent)
- **Legal Documents:** Will, healthcare proxy, POA (encrypted, shareable with
  attorney)
- **Photo Memories:** Family photo sharing (end-to-end encrypted)

## What We Will NOT Build

- Advertising platform (no data selling)
- Location tracking (no where-have-you-been history)
- Social network (no friend suggestions or algorithmic feeds)
- Health prediction AI (no selling insights to pharma)

## How You Can Influence This Roadmap

Open an issue on Github — We listen to users and their family members.
